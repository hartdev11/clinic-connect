# E5.10 — Knowledge Washing Machine (Design only)

## สถานะ

- **Enterprise only**
- **Feature flag** — เปิดได้เฉพาะ plan enterprise
- **❌ ไม่ implement Phase 1**

---

## Concept

Knowledge Washing Machine = pipeline ที่ล้าง/ปรับปรุง knowledge ก่อนหรือหลัง ingest เพื่อให้ข้อมูลสอดคล้องและคุณภาพสูง

## Feature flag

| Flag | Default | หมายเหตุ |
|------|---------|----------|
| `KNOWLEDGE_WASHING_MACHINE_ENABLED` | false | เปิดเฉพาะ org ที่มี plan = enterprise |
| ตรวจที่ org-level | plan === "enterprise" | ถ้าไม่ใช่ enterprise → bypass |

## ขั้นตอน (concept)

1. **Input** — Knowledge doc ก่อน/หลัง save
2. **Normalize** — ปรับ format, ลบข้อความซ้ำซ้อน
3. **Validate** — ตรวจความครบถ้วน, coherence
4. **Enrich** — (optional) เพิ่ม metadata, แนะนำ category
5. **Output** — doc ที่ผ่านการล้างแล้ว

## Phase 2+ scope

- ไม่ implement ใน Phase 1
- เตรียม design สำหรับ Phase 2 เมื่อมี Enterprise plan
